# Log-Ingestor
### Requirements

- Java 8 or higher
- Maven

### Installation

1. Clone this repository:

   ```sh
   git clone https://github.com/adityavag/Log-Ingestor.git
2. Build the poject using Maven:

   ```sh
   mvn clean install
3. Run the application:
   ```sh
   mvn spring-boot:run
